# 📊 Topic Overview: Arrays & Strings

Arrays and Strings are the foundational blocks of DSA. Mastery here involves pattern recognition for contiguous data.

## 🔑 Key Concepts Checklist
- [ ] **Two Pointers:** Opposite ends (Sorted search) or same direction (In-place modification).
- [ ] **Sliding Window:** Fixed size (Subarray sum) or Variable size (Longest substring).
- [ ] **Prefix Sum / Difference Array:** Range queries and range updates.
- [ ] **Hashing:** Frequency counting and O(1) lookups.
- [ ] **Kadane's Algorithm:** Maximum subarray sum.
- [ ] **Dutch National Flag:** Three-way partitioning.

## 🎯 Essential Problem Checklist (95% Coverage)
| Problem | Key Concept | Difficulty |
| :--- | :--- | :--- |
| **Two Sum** | Hash Map | Easy |
| **Best Time to Buy/Sell Stock** | Kadane's / One-pass | Easy |
| **Contains Duplicate** | Hash Set | Easy |
| **Product of Array Except Self** | Prefix/Suffix arrays | Medium |
| **Maximum Subarray** | Kadane's Algorithm | Medium |
| **3Sum** | Two Pointers (Sorted) | Medium |
| **Longest Substring Without Repeating** | Sliding Window | Medium |
| **Group Anagrams** | Hashing | Medium |
| **Valid Palindrome** | Two Pointers | Easy |
| **Valid Anagram** | Hashing | Easy |
| **Trapping Rain Water** | Two Pointers / Monotonic Stack | Hard |
| **Minimum Window Substring** | Sliding Window | Hard |

## ⏱️ Complexity Cheatsheet
| Technique | Time Complexity | Space Complexity |
| :--- | :--- | :--- |
| Two Pointers | $O(N)$ | $O(1)$ |
| Sliding Window | $O(N)$ | $O(1)$ or $O(K)$ |
| Prefix Sum | $O(N)$ build, $O(1)$ query | $O(N)$ |
| Hashing | $O(N)$ | $O(N)$ |
| Sorting + 2 Pointers | $O(N \log N)$ | $O(1)$ |
